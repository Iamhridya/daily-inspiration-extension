# Daily Inspiration Chrome Extension

Daily Inspiration is a calming Chrome extension that displays motivational quotes in a beautiful purple wellness-themed popup.

## ✨ Features

- Fresh motivational quotes
- Save favorite quotes
- Elegant butterflies and sparkle animations
- Fast lightweight popup
- Beautiful feminine calming UI

## 🚀 Install Locally

1. Download this repo
2. Open Chrome
3. Visit chrome://extensions
4. Enable Developer Mode
5. Click Load Unpacked
6. Select project folder
